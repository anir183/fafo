# kotlin multiplatform

* does not work without intellij or android
* needs Kotlin Multiplatform and org.jetbrains.android plugins
* min ide version 2025.3.2 needed
* uses old gradle 8.4.smth
* does not work with java 25.. max java 24
* takes a shit ton of time to setup.. downloads lots of android and other stuff
* does not work reproducably in different devices
* can be pretty laggy on decent hardware
* does not work with kotlin-lsp tooling ([issue#3](https://github.com/Kotlin/kotlin-lsp/issues/3))
* no cli or tools to create project structure (have to use [this](https://kmp.jetbrains.com/?android=true&ios=true&iosui=compose&includeTests=true))
